◆ compare_learning_methods()

void ml::machine_learning_interface::compare_learning_methods ( )

Compare different learning methods and return the best one.

If alpha < 0.01, only a surrogate model for evacuations is trained. If alpha > 0.99 only a surrogate model for travel between consecutive events is trained. Otherwise, both surrogate models are trained.

Parameters
alphaThe weight of both objective function elements, viz. evacuations and travels between consecutive events.
Returns
The best learning method.

void ml::machine_learning_interface::train ( )

Trains the surrogates for the required objective functions given the choice of learning method.

If alpha < 0.01, only a surrogate model for evacuations is trained. If alpha > 0.99 only a surrogate model for travel between consecutive events is trained. Otherwise, both surrogate models are trained.

alphaThe weight of both objective function elements, viz. evacuations and travels between consecutive events.
lmThe learning method used in the surrogates.
